>> SFIS/MES System

        MES即制造执行系统(manufacturing execution system，简称MES）是美国AMR公司(Advanced Manufacturing Research，Inc．)在90年代初提出的，旨在加强MRP计划的执行功能，把MRP计划同车间作业现场控制，通过执行系统联系起来。这里的现场控制包括PLC程控器、数据采集器、条形码、各种计量及检测仪器、机械手等。MES系统设置了必要的接口，与提供生产现场控制设施的厂商建立合作关系。

>> 程式权限介绍

       程式权限是针对程式安全性与多元性考虑设立的一种权限管控，一个程式可能涉及多方面的技能、操作与查询，不同的单位有不同的使用页面(都在同一个程式下)，就需要针对性的开立对应的权限，防止错误使用非本单位页面导致生产异常发生。
